# next-.net-portfolio
Production Full Stack Portfolio

A modern full-stack portfolio demonstrating production-level web development using Next.js, ASP.NET Core Web API, PostgreSQL, and cloud deployment with Vercel and Render.

| Service     | URL                                                                                                      |
| ----------- | -------------------------------------------------------------------------------------------------------- |
| Frontend    | [https://next-net-portfolio-vercel.vercel.app](https://next-net-portfolio-vercel.vercel.app)             |
| Backend API | [https://siwarut-portfolio-api.onrender.com](https://siwarut-portfolio-api.onrender.com)                 |
| Swagger     | [https://siwarut-portfolio-api.onrender.com/swagger](https://siwarut-portfolio-api.onrender.com/swagger) |
